# Employee Management System using Python and SQLite

A console-based application to manage employee records, perform CRUD operations, 
and automate payroll and departmental reporting using SQLite.

## Features
- Add, view, update, search, and delete employee details  
- Department-wise report generation  
- Salary update and automation  
- Input validation and error handling  

## Tech Stack
- **Language:** Python  
- **Database:** SQLite  
- **Tools:** VS Code, Git, GitHub

   ```bash
